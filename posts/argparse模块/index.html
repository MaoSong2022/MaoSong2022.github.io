argparse module

Posted Oct 30, 2022 Updated Oct 30, 2022
By Mao Song
6 min read

一句话概括，argparse是一个支持在命令行中给python程序输入参数的一个模块。 举一个例子，如果我们有一个文件计算并输出一个数的平方，我们可以这样做：

# main.py
def main() -> None:
	num = 4
	print(num**2)

if __name__ == '__main__':
	main()

这样做的缺点是，我们每次计算时，都需要更改程序来重新进行计算。这时候argparse就可以用来给main函数传递参数来实现这一功能。 我们首先给出argparse版本：

# main.py
import argparse

def main(args) -> None:
	num = args.integer
	print(num**2)

if __name__ == '__main__':
	parser = argparse.ArgumentParser(description='input an integer and print the square of the number.')
	parser.add_argument('integer', type=int, help='a number')
	args = parser.parse_args()
	main(args)

然后我们在命令行中输入

1

python main.py 4

就可以得到输出

16

这里的4就是我们自己设置的输入，我们可以随意更改。我们可以使用-h或者--help参数来查看我们可以输入的命令行参数

可以看到，输出分为以下几部分:

  • line 1: 输出了运行main.py的方式()

  • line 5-6: 我们必须给定的参数(positional arguments), 首先是这个参数的名称integer，然后是我们对这个参数的解释a number
  • line 8-9: 可选参数(optional arguments)，定义与上面类似，不同点在于可选参数一般有默认值，当我们不给定具体值的时候，可选参数使用默认值（一般为None）。

可选参数

为了了解可选参数(optional arguments)，我们将平方操作扩展为一般的n次方操作，如果n不给定的话，我们就计算平方数，为此，我们可以这样写：

# main.py
import argparse

def main(args) -> None:
	num = args.integer
	power = args.power
	print(num**power)

if __name__ == '__main__':
	parser = argparse.ArgumentParser(description='input an integer and a power and print the power of the number.')
	parser.add_argument('integer', type=int, help='a number')
	parser.add_argument('--power', type=int, 
						help='power of a number', 
						default=2)
	args = parser.parse_args()
	main(args)

Action

那么，如果有时候我们想要一些更详细的debug信息该怎么办呢，一种普通的做法就是我们增加一个变量，变量为不同值的时候代表不同层级的debug信息。这时，我们就可以使用action来实现这个开关的作用。

import argparse

def main(args) -> None:
	num = args.integer
	power = args.power
	if (args.verbose):
		print(f'the number {num} to the power of {power} is {num**power}')
	else:
		print(num**power)

if __name__ == '__main__':
	parser = argparse.ArgumentParser(description='input an integer and a power and print the power of the number.')
	parser.add_argument('integer', type=int, help='a number')
	parser.add_argument('--power', type=int,,
						help='power of a number', 
						default=2)
	parser.add_argument('--verbose', action='store_false')
	args = parser.parse_args()
	main(args)

此时，我们运行得到结果：

1
2

python main.py --power 3 4
# the number 4 to the power of 3 is 64

1
2

python main.py --power 3 5 --verbose
# 64

这样我们就达到了控制debug信息输出的目的。对于更高级的用法，我们可以参考argparse。

import argparse

def main(args) -> None:
  num = args.integer
  power = args.power
  if (args.verbose):
    print(f'the number {num} to the power of {power} is {num**power}')
  else:
    print(num**power)

if __name__ == '__main__':
  parser = argparse.ArgumentParser(description='input an number as base and (possibly) a number as power and print the power of the number.')
  parser.add_argument('integer', type=int, help='base')
  parser.add_argument('-p', '--power', type=int, 
            help='exponent', 
            default=2)
  parser.add_argument('-v', '--verbose', action='store_false')
  args = parser.parse_args()
  main(args)
