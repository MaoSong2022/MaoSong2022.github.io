## Introduction

Evaluation 是为了评估模型的表现，进而：

1. 判别模型是否正在进步
2. 选择合适的模型
3. 评估是否可以投入生产

形式：

- closed form: 自动化评估，如 MMLU. 问题在于对于 prompt 比较敏感，可能会出现 contamination
- free form: 开放答案
- llm as judge

一个好的 benchmark:

1. what is the goal/ what to evaluate
2. what is the task / what is an env to run the agent to achieve the goal
3. how to evaluate the agent? how to build the data collection pipeline

Principle：

1. real-world
2. have different difficulty levels
3. not easy to get saturated or contaminated

good benchmarks

1. cybergym
2. tau-bench
