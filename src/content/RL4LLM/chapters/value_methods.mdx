## Value-Based Methods

在 [policy evaluation](#policy_evaluation) 一节中，我们介绍了在固定策略 $\pi$ 的情况下，如何求解对应的 value function 或者 Q function.
我们虽然给出了一个 policy iteration algorithm, 但是其严重依赖初始策略 $\pi^0$.
为了解决这个问题，我们可以想办法直接找到最优策略 $\pi^*$ 对应的 optimal value function $V^*$ 或者 optimal Q function $V^*$.
这也是 Value-based methods 的核心思想：

<Note variant="info">

求解得到 optimal value function $V^*$ 或者 optimal Q function $Q^*$, 再得到 optimal policy $\pi^*$.

</Note>

在 Bellman Equation 中，我们介绍了 value iteration 的基本思路：通过 Bellman optimality operator 迭代求解 $V^*$ 和 $Q^*$.
本节将 value iteration 的思想与函数逼近、TD learning 等技术结合，得到实用的 value-based 算法.

在 Model-Free 设定下，我们无法直接计算转移概率 $\mathbb{E}_{s' \sim p}$。
我们必须通过与环境交互产生的轨迹样本 $(s, a, r, s', a')$ 来更新价值。

#### SARSA

SARSA 的核心在于它的更新动作 $a'$ 严格来自于当前的 $\epsilon$-greedy 策略：

$$\boxed{Q(s, a) \leftarrow Q(s, a) + \alpha \left[ r + \gamma Q(s', a') - Q(s, a) \right]}$$

#### Q-Learning (Off-Policy TD Control)

与 SARSA 不同，Q-Learning 在估计未来价值时，直接贪心地选择最优动作，而不受当前探索策略的影响：

$$\boxed{Q(s, a) \leftarrow Q(s, a) + \alpha \left[ r + \gamma \max_{a' \in \mathcal{A}} Q(s', a') - Q(s, a) \right]}$$

### Function Approximation

当状态空间较大或连续时，我们需要使用函数逼近，与上一节类似，我们用 $V_{\phi}$ 近似 $V^{\pi}$:

$$
J(\phi) = \mathbb{E}_{s}\left[\frac{1}{2}\left(V^{\pi}(s) - V_{\phi}(s)\right)^2\right]
$$

目标函数的梯度为：

$$
\nabla_{\phi} J(\phi) = -\mathbb{E}_{s}\left[(V^{\pi}(s) - V_{\phi}(s))\nabla_{\phi} V_{\phi}(s)\right]
$$

随机梯度下降的更新形式为：

$$
\phi_{k+1} = \phi_k + \alpha_k (V^{\pi}(s) - V_{\phi}(s))\nabla_{\phi} V_{\phi}(s)
$$

$V^{\pi}(s)$ 可以通过两种方式来近似：

1. **MC 近似**: $V^{\pi}(s) = \mathbb{E}^{\pi}[G_0 \mid s_0 = s] \approx G_0$
2. **TD 近似**: $V^{\pi}(s) \approx r_t + \gamma V_{\phi}(s_{t+1})$

### Deep Q-Network (DQN)

在 TODO 的基础上，DQN 进一步优化了训练的目标函数，用于逼近 optimal Q function:

$$
\mathcal{L}(\phi) = \mathbb{E}_{(s,a,r,s') \sim \mathcal{D}} \left[ \frac{1}{2} \left( r + \gamma \max_{a'} Q_{\phi^-}(s', a') - Q_{\phi}(s, a) \right)^2 \right]
$$

其中 $\phi^-$ 为**目标网络（Target Network）**的参数，用于切断数据自举（Bootstrapping）带来的训练正反馈爆炸。
为了保证求导时 Target 的稳定性，我们必须在代码或公式里明确使用 `stop-gradient` 算子：

$$y = r + \gamma \max_{a'} \mathrm{sg}\left[Q_{\phi^-}(s', a')\right]$$

DQN 针对 optimal policy 应用了 TD learning, 其目标函数为：

$$
\min_{\phi} \mathcal{L}(\phi) = \mathbb{E}_{(s, a, r, s') \sim \mathcal{D}}\left[\frac{1}{2}\left(Q_{\phi}(s, a) - \left(r + \gamma \max_{a'} Q_{\phi}(s', a')\right)\right)^2\right]
$$

这里 $\mathcal{D}$ 是 replay buffer, 存储了过去的 transition $(s, a, r, s')$.

<Note variant="info" emoji="💡">

DQN 的两个关键技术：

1. **Experience Replay**: 使用 replay buffer 存储过去的 transitions，从中随机采样进行训练。这打破了数据之间的相关性，提高了样本效率.
2. **Target Network**: 使用一个独立的 target network $Q_{\phi^{-}}$ 来计算 TD target, 其参数 $\phi^{-}$ 定期从 $\phi$ 复制（或通过 Polyak averaging 更新）。这稳定了训练过程.

</Note>

带有 target network 和 stop-gradient 的 DQN 目标函数为：

$$
\mathcal{L}(\phi) = \mathbb{E}_{(s, a, r, s') \sim \mathcal{D}}\left[\frac{1}{2}\left(Q_{\phi}(s, a) - \left(r + \gamma \, \max_{a'} \mathrm{sg}[Q_{\phi^{-}}(s', a')]\right)\right)^2\right]
$$

对应的 Python 实现：

```python
# Compute current Q-values
q_values = Q_phi(states).gather(1, actions)

# Compute target Q-values with target network
with torch.no_grad():
    next_q_values = Q_target(next_states).max(1)[0]
    targets = rewards + gamma * next_q_values * (1 - dones)

# Compute loss
loss = 0.5 * (q_values - targets) ** 2
loss = loss.mean()
```

<Algorithm title="DQN with Experience Replay">

1. Initialize replay buffer $\mathcal{D}$ with capacity $N$.
2. Initialize Q-network $Q_{\phi}$ with random weights.
3. Initialize target network $Q_{\phi^{-}}$ with weights $\phi^{-} = \phi$.
4. **for** episode = $1, \dots, M$:
   - $s_0 \sim p_0$.
   - **for** $t = 0, \dots, T-1$:
     - Select action with $\epsilon$-greedy:
       $$
       a_t = \begin{cases}
       \arg\max_a Q_{\phi}(s_t, a), & \text{with prob } 1 - \epsilon \\
       \text{random action},         & \text{with prob } \epsilon
       \end{cases}
       $$
     - $(r_t, s_{t+1}) \sim p(\cdot, \cdot \mid s_t, a_t)$.
     - Store $(s_t, a_t, r_t, s_{t+1})$ in $\mathcal{D}$.
     - Sample random minibatch $\{(s_i, a_i, r_i, s'_i)\}$ from $\mathcal{D}$.
     - Compute target:
       $$
       y_i = \begin{cases}
       r_i + \gamma \max_{a'} Q_{\phi^{-}}(s'_i, a'), & \text{if not terminal} \\
       r_i,                                              & \text{if terminal}
       \end{cases}
       $$
     - Update Q-network:
       $$
       \mathcal{L} = \frac{1}{|\text{batch}|} \sum_i \frac{1}{2} (Q_{\phi}(s_i, a_i) - y_i)^2, \quad \phi \gets \phi - \alpha \nabla_{\phi} \mathcal{L}
       $$
     - Every $C$ steps: $\phi^{-} \gets \phi$.

</Algorithm>
