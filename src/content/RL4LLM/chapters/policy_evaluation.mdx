## Policy Evaluation

在 [Bellman Equation Theorem](##bellman-equation-theorem) 一节中，我们已经知道 $V^{\pi}$ 满足 Bellman equation.
在本节中，我们将关注

<Note variant="info">

给定一个策略 $\pi$，如何高效求解策略 $\pi$ 对应的 value function $V^\pi$?

</Note>

我们将从上一节介绍的 [value iteration](#value-iteration) 开始，一步步进行优化，最终扩展到现代 RL 算法所使用的方法。

### A ToyLLM Environment

后续章节中，我们将使用一个非常简单的 `ToyLLM` 环境来对比不同的方法。

`ToyLLM` 的定义与 [Connection with LLMs](#connection-with-llm) 一致，我们给定一个词表 $V$, 一个 target sequence $s$, 我们的 policy model 从 `<bos>` token 开始，每次生成一个新的 token 直到采样到 `<eos>` token 或者到达 `max_len` 形成一个 rollout $\tau$, 最终的 return 定义为

$$
R(\tau)=\begin{cases}
   1.0,&\text{ if }\tau = s \\
   0,&\text{ otherwise}
\end{cases}
$$

因此，我们的 action space $\mathcal{A}=V$, $\mathcal{S}=V^T$, 这里 $T$ 是最大输出长度 `max_len`. 为了后续表达方便，我们用 $N:=|V|^T$ 来表示状态空间大小。

### Revisiting Value Iteration

我们在上一节介绍了一个基于 contraction mapping 求解 value function 的方法，对于离散的 state space 来说，我们可以将 [Bellman Equation](#bellman_equation_value) 写为如下的矩阵形式

$$
V_{\pi} = R_{\pi} + \gamma P_{\pi} V_{\pi}
$$

其中

$$
\begin{aligned}
   V_\pi &= [V_{\pi}(s_1),\dots, V_{\pi}(s_N)]^T\in\mathbb{R}^N\\
   R_\pi &=[R_{\pi}(s_1),\dots, R_{\pi}(s_N)]^T\in\mathbb{R}^N\\
   P_\pi &=[P_\pi(s_j\mid s_i)]\in\mathbb{R}^{N\times N}
\end{aligned}
$$

可以看到，上面这个其实是一个线性方程组，即

$$
(\mathbf{I}_N - \gamma P_{\pi})V_{\pi}  = R_{\pi}
$$

对于这个线性方程组，我们可以用矩阵求逆或者迭代解法求解。
算法如下所示

<Algorithm title="Value function Matrix Exact Solve">

**Input**: policy $\pi$, discount factor $\gamma$, state space $\mathcal{S}$ with size $N$

1. assign index $i(s)$ for $s\in\mathcal{S}, i=1,\dots,N$
2. $P=\mathbf{0}_{N\times N}$, $R=\mathbf{0}_{N}$
3. **For** each $s\in\mathcal{S}$
   1. $i\gets i(s)$
   2. **For** each $a\in\mathcal{A}$
      - $(s', r, \mathrm{done}) \gets \mathrm{STEP}(s, a)$
      - $R[i]\gets R[i]+\pi(a\mid s)\cdot r$
      - **If** NOT $\mathrm{done}$
        - $j\gets i(s')$
        - $P[i,j]\gets P[i,j]+\pi(a\mid s)$
4. $V\gets \mathrm{SOLVE}(I-\gamma P, R)$
5. return $V$

</Algorithm>

<Algorithm title="Value function Matrix Iterative Solve">

**Input**: policy $\pi$, discount factor $\gamma$, convergence threshold $\theta$, state space $\mathcal{S}$

1. $V(s)=0, s\in\mathcal{S}$
2. **While** $\Delta > \theta$:
   1. $\Delta\gets0$
   2. **For** each $s\in\mathcal{S}$
      1. $v\gets0$
      2. **For** each $a\in\mathcal{A}$
         - $(s', r, \mathrm{done}) \gets \mathrm{STEP}(s, a)$
         - **If** $\mathrm{done}$
           - $v \gets v + \pi(a|s) \cdot r$
         - **Else**
           - $v \gets v + \pi(a|s) \cdot (r + \gamma \cdot V(s'))$
      3. $\Delta ← max(\Delta, |v − V(s)|)$
      4. $V(s) \gets v$
3. return $V$

</Algorithm>

可以看到，随着词表大小 $N$ 的增加， 因为矩阵求逆是一个 $\mathcal{O}(N^3)$ 复杂度的算法，其所需时间急剧上升。
而迭代解法虽然比矩阵求逆效率更高，但是仍然需要枚举全部状态，当状空间增大之后，迭代解法的时间也无法接受。

为了解决这个问题，我们会利用 Bellman equation、采样和函数逼近，高效地估计或拟合给定策略对应的 value function $V^{\pi}$.

### Monte Carlo

首先，由 $V^{\pi}$ 定义，我们有：

$$
V^{\pi}(s_0) = \mathbb{E}^{\pi}\left[R(\tau) \mid s_0\right]
$$

这是一个随机变量，由我们可以利用 Monte Carlo (MC) 方法来得到一个 [Unbiased Estimator](https://maosong.website/blog/math/#probability)，即从 $s_0$ 出发，独立随机采样 (i.i.d.) $N$ 条轨迹：

$$
\{\tau^{(i)} = (s_0, a_0^{(i)}, r_0^{(i)}, \dots, a_{T^{(i)}-1}^{(i)}, r_{T^{(i)}-1}^{(i)}, s_{T^{(i)}}^{(i)})\}, \quad i = 1, \dots, N
$$

然后我们使用样本平均来近似期望：

$$
V^{\pi}(s_0) = \mathbb{E}^{\pi}[R(\tau) \mid s_0] \approx \frac{1}{N} \sum_{i=1}^{N} R(\tau^{(i)})
$$

如果 $\mathcal{S}$ 比较小，我们可以遍历 $s \in \mathcal{S}$ 来估计 $V^{\pi}$. 但是，如果 $\mathcal{S}$ 比较大，这种方法的计算复杂度较高.

<Note variant="info" emoji="💡">

- **优点**: 简单、直观、无需环境模型
- **缺点**: 必须等轨迹结束后才能得到一次完整的估计

</Note>

### Function Approximation

MC 方法虽然简单直观，但是其适用于$\mathcal{S}$ 比较小的场景。
当状态空间 $\mathcal{S}$ 较大或连续时，我们可以使用函数逼近的方法来近似 $V^{\pi}$.
这里我们使用 $V_{\phi}$ 来近似 $V^{\pi}$, 其中 $\phi$ 是函数的参数，我们使用最小二乘误差作为损失函数：

$$
\min_{\phi} \mathcal{L}(\phi) = \mathbb{E}_{s_0 \sim p_0}^{\pi}\left[\frac{1}{2}\left(V_{\phi}(s_0) - V^{\pi}(s_0)\right)^2\right]
$$

对目标函数求导得到：

$$
\begin{aligned}
\nabla_{\phi} \mathcal{L}(\phi) &= \mathbb{E}_{s_0 \sim p_0}^{\pi}\left[\left(V_{\phi}(s_0) - V^{\pi}(s_0)\right)\nabla_{\phi} V_{\phi}(s_0)\right] \\
&= \mathbb{E}_{s_0 \sim p_0}^{\pi}\left[\left(V_{\phi}(s_0) - \mathbb{E}^{\pi}[R(\tau) \mid s_0]\right)\nabla_{\phi} V_{\phi}(s)\right] \\
&= \mathbb{E}_{s_0 \sim p_0}^{\pi}\left[\mathbb{E}^{\pi}\left[\left(V_{\phi}(s_0) - R(\tau)\right)\nabla_{\phi} V_{\phi}(s_0) \mid s_0\right]\right] \\
&= \mathbb{E}_{s_0 \sim p_0}^{\pi}\left[\left(V_{\phi}(s_0) - R(\tau)\right)\nabla_{\phi} V_{\phi}(s_0) \mid s_0\right]
\end{aligned}
$$

现在我们可以使用 MC 方法来估计梯度：

$$
\nabla_{\phi} \mathcal{L}(\phi) \approx g := \frac{1}{N}\sum_{i=1}^{N} \left(V_{\phi}(s^{(i)}) - R(\tau^{(i)})\right)\nabla_{\phi} V_{\phi}(s^{(i)})
$$

结合 MC 和 SGD 的算法如下：

<Algorithm title="Value function approximation with MC">

**while** not converged:

1. $s_0 \sim p_0$, $t = 0$.
2. **while** $s_t \neq \langle\text{term}\rangle$:
   - $a_t \sim \pi(\cdot \mid s_t)$
   - $(r_t, s_{t+1}) \sim p(\cdot, \cdot \mid s_t, a_t)$
   - $t \gets t + 1$
3. Set $T = t$.
4. $g \gets \left(V_{\phi}(s_0) - \sum_{t=0}^{T-1} \gamma^t r_t\right)\nabla_{\phi} V_{\phi}(s_0)$.
5. Update $\phi$ using $g$ with an optimizer.

</Algorithm>

### Temporal Difference Learning

MC 的优点是梯度是一个无偏估计，其缺点在于我们必须完整采样完一条轨迹才能得到这个估计，对于 long-horizon task, MC 方法效率很低。
为了解决这个问题，我们可以使用 Temporal difference (TD) learning 方法。

Temporal difference (TD) learning 通过相邻两步的转换关系来近似 value function. 其核心思想在于

<Note variant="info">通过新的信息和新的预测来更新旧的预测</Note>

注意到：

$$
V^{\pi}(s_t) = \mathbb{E}^{\pi}[r_t + \gamma V^{\pi}(s_{t+1}) \mid S_t=s_t]
$$

我们定义 TD target 和 TD error 如下

<Definition term='TD target and TD error'>

对于 value function $V^{\pi}(s_t)$, 我们定义 **TD target** 为:

$$
\overline{V}_t = r_t + \gamma V^{\pi}(s_{t+1})
$$

定义 **TD error** 为

$$
\delta_t = V^{\pi}(s_t) - \left(r_t + \gamma V^{\pi}(s_{t+1})\right)
$$

</Definition>

$$
\min_{\phi} \mathcal{L}(\phi) = \mathbb{E}_{s_0 \sim p_0}^\pi\left[\frac{1}{2}\left(V_{\phi}(s_0) - r_0 - \gamma V^{\pi}(s_1)\right)^2\right]
$$

对应的梯度为：

$$
\nabla_{\phi} \mathcal{L}(\phi) = \mathbb{E}_{s_0 \sim p_0}^{\pi}\left[\left(V_{\phi}(s_0) - r_0 - \gamma \boxed{V^{\pi}(s_1)}\right)\nabla_{\phi} V_{\phi}(s_0) \mid s_0\right]
$$

这里梯度力包含了一个未知量 $V^{\pi}(s_1)$. 为了解决这个问题，一个自然的想法是使用当前的 value function $V_{\phi}(s_1)$ 来进行代替：

$$
\nabla_{\phi} \mathcal{L}(\phi) \approx \mathbb{E}_{s_0 \sim p_0}^{\pi}\left[\left(V_{\phi}(s_0) - r_0 - \gamma V_{\phi}(s_1)\right)\nabla_{\phi} V_{\phi}(s_0) \mid s_0\right]
$$

这样，我们就得到了一个结合 TD 和 SGD 的 naive TD learning 算法：

<Algorithm title="Value function approximation with TD (not correct)">

**while** not converged:

1. $s_0 \sim p_0$, $a_0 \sim \pi(\cdot \mid s_0)$, $(r_0, s_1) \sim p(\cdot, \cdot \mid s_0, a_0)$.
2. computer TD target
   $$
   y = \begin{cases}
   r_0 + \gamma V_{\phi}(s_1), & \text{if } s_1 \neq \langle\text{term}\rangle \\
   r_0,                        & \text{otherwise}
   \end{cases}
   $$
3. $g = (V_{\phi}(s_0) - y) \nabla_{\phi} V_{\phi}(s_0)$.
4. Update $\phi$ using $g$ with an optimizer.

</Algorithm>

我们来看一下上述算法对应的 PyTorch 实现：

```python
pred = V_phi(s0)
target = r + gamma * V_phi(s1)
td_error = pred - target
loss = 0.5 * td_error ** 2
loss.backward()
```

可以看到，我们实际上求的梯度是：

$$
\begin{aligned}
&\nabla_{\phi}\left[\frac{1}{2}\left(V_{\phi}(s) - (r + \gamma V_{\phi}(s'))\right)^2\right] \\
&= (V_{\phi}(s) - (r + \gamma V_{\phi}(s')))(\nabla_{\phi} V_{\phi}(s) - \nabla_{\phi} V_{\phi}(s'))\\
&\neq g
\end{aligned}
$$

这显然与 $g$ 不相等.

为了解决这个问题，我们可以使用 **stop-gradient** 技巧来避免 $V_{\phi}(s')$ 参与反向传播，此时目标函数变为：

$$
\mathcal{L}(\phi)=\frac{1}{2}\left(V_{\phi}(s) - (r + \gamma \, \mathrm{sg}[V_{\phi}(s')])\right)^2
$$

其中 $\mathrm{sg}[\cdot]$ 是 stop-gradient operator, 满足：

$$
\mathrm{sg}[x] = \begin{cases}
x & \text{forward pass} \\
0 & \text{backward pass}
\end{cases}
$$

对应的梯度就是：

$$
\nabla_{\phi}\left[\frac{1}{2}\left(V_{\phi}(s) - (r + \gamma \, \mathrm{sg}[V_{\phi}(s')])\right)^2\right] = (V_{\phi}(s) - (r + \gamma V_{\phi}(s')))\nabla_{\phi} V_{\phi}(s) = g
$$

对应的 Python 代码为：

```python
pred = V_phi(s0)
target = r + gamma * V_phi(s1)
td_error = pred - target.detach()  # stop-gradient operator
loss = 0.5 * td_error ** 2
loss.backward()
```

最终，正确的 TD 算法如下：

<Algorithm title="Value function approximation with TD">

**while** not converged:

1. $s_0 \sim p_0$, $a_0 \sim \pi(\cdot \mid s_0)$, $(r_0, s_1) \sim p(\cdot, \cdot \mid s_0, a_0)$.
2. computer TD target
   $$
   y = \begin{cases}
   r_0 + \gamma \, \mathrm{sg}[V_{\phi}(s_1)], & \text{if } s_1 \neq \langle\text{term}\rangle \\
   r_0,                                         & \text{otherwise}
   \end{cases}
   $$
3. $g = (V_{\phi}(s_0) - y) \nabla_{\phi} V_{\phi}(s_0)$.
4. Update $\phi$ using $g$ with an optimizer.

</Algorithm>

<Note variant="warn" emoji="⚠️">
  上述做法其实是 **semi-gradient methods**,
  这类方法在参数更新时，只计算部分梯度。虽然这种方法牺牲了严格的梯度下降性质，但是其能够保证更好的表现.
</Note>

### k-step TD

实际上，我们可以进一步推广 TD 到多步场景，注意到：

$$
\begin{aligned}
V^{\pi}(s_0) &= \mathbb{E}^{\pi}[r_0 + \gamma V^{\pi}(s_1) \mid s_0] \\
&= \mathbb{E}^{\pi}[r_0 + \gamma \mathbb{E}^{\pi}[r_1 + \gamma V^{\pi}(s_2) \mid s_1] \mid s_0] \\
&= \mathbb{E}^{\pi}[\mathbb{E}^{\pi}[r_0 + \gamma r_1 + \gamma^2 V^{\pi}(s_2) \mid s_1] \mid s_0]
\end{aligned}
$$

由 [Law of Total Expectation](https://maosong.website/blog/math/#law-of-total-expectation):

$$
\mathbb{E}^{\pi}[\mathbb{E}^{\pi}[X \mid s_1] \mid s_0 = s] = \mathbb{E}^{\pi}[X \mid s_0]
$$

因此：

$$
V^{\pi}(s_0) = \mathbb{E}^{\pi}[r_0 + \gamma r_1 + \gamma^2 V^{\pi}(s_2) \mid s_0]
$$

重复这个过程 k 次，我们就可以得到 k 步展开：

$$
V^{\pi}(s_0) = \mathbb{E}^{\pi}\left[\sum_{i=0}^{k-1} \gamma^i r_i + \gamma^k V^{\pi}(s_k) \mid s_0\right]
$$

基于 k-step transition ，我们可以构建对应的目标函数并应用 stop-gradient：

<Algorithm title="Value function approximation with k-step TD">

**while** not converged:

1. $s_0 \sim p_0$, set $t = 0$.
2. **for** $t = 0, \ldots, k-1$:
   - $a_t \sim \pi(\cdot \mid s_t)$
   - $(r_t, s_{t+1}) \sim p(\cdot, \cdot \mid s_t, a_t)$
3. compute k-step TD target
   $$
   y = \begin{cases}
   \sum_{t=0}^{k-1} \gamma^t r_t + \gamma^k \, \mathrm{sg}[V_{\phi}(s_k)], & \text{if } s_k \neq \langle\text{term}\rangle \\
   \sum_{t=0}^{k-1} \gamma^t r_t, & \text{otherwise}
   \end{cases}
   $$
4. $g \gets (V_{\phi}(s_0) - y) \nabla_{\phi} V_{\phi}(s_0)$.
5. Update $\phi$ using $g$ with an optimizer.

</Algorithm>

一般来说，$k = 5$ 是一个常用的选择.

### MC v.s. TD

| 方法 | 特点                                            |
| :--: | :---------------------------------------------- |
|  MC  | 需要完整的一次采样，无偏但 variance 高          |
|  TD  | 通过 bootstrap 提高采样效率，有偏但 variance 低 |

TD 本质上是 bootstrap: 当需要使用 $V^{\pi}$ 时，用 $V_{\phi}$ 来代替.
当 $V_{\phi}$ 与 $V^{\pi}$ 差距较大时，bootstrap 会导致训练不稳定，而 MC 则相对稳定.

### Q-Function Evaluation

对于 Q-function, 我们也可以设计类似的算法.

对于 MC, 有：

$$
Q^{\pi}(s_0, a_0) = \mathbb{E}^{\pi}\left[\sum_{t=0}^{T-1} \gamma^t r_t \mid s_0, a_0 \right] \approx \frac{1}{N} \sum_{i=1}^{N} \sum_{t=0}^{T^{(i)}-1} \gamma^t r_t^{(i)}
$$

当使用模型来近似时，目标函数为：

$$
\min_{\phi} \mathcal{L}(\phi) = \mathbb{E}_{s_0 \sim p_0, \, a_0 \sim \pi(\cdot \mid s_0)}\left[\frac{1}{2}\left(Q_{\phi}(s_0, a_0) - Q^{\pi}(s_0, a_0)\right)^2\right]
$$

对应的梯度：

$$
\nabla_{\phi} \mathcal{L}(\phi) = \mathbb{E}_{s_0 \sim p_0, \, a_0 \sim \pi(\cdot \mid s_0)}^{\pi}\left[\left(Q_{\phi}(s_0, a_0) - \sum_{t=0}^{T-1} \gamma^t r_t\right)\nabla_{\phi} Q_{\phi}(s_0, a_0)\right]
$$

对应的算法如下

<Algorithm title="Q-function approximation with MC">

**while** not converged:

1. $s_0 \sim p_0$, $a_0 \sim \pi(\cdot \mid s_0)$, set $t = 0$.
2. **while** $s_t \neq \langle\text{term}\rangle$:
   - $a_t \sim \pi(\cdot \mid s_t)$
   - $(r_t, s_{t+1}) \sim p(\cdot, \cdot \mid s_t, a_t)$
   - $t \gets t + 1$
3. Set $T = t$.
4. $g = \left(Q_{\phi}(s_0, a_0) - \sum_{t=0}^{T-1} \gamma^t r_t\right) \nabla_{\phi} Q_{\phi}(s_0, a_0)$.
5. Update $\phi$ using $g$ with an optimizer.

</Algorithm>

对于TD learning, 与 value function 的算法类似，我们也由对应版本的算法：

<Algorithm title="Q-function approximation with TD">

**while** not converged:

1. $s_0 \sim p_0$, $a_0 \sim \pi(\cdot \mid s_0)$, $(r_0, s_1) \sim p(\cdot, \cdot \mid s_0, a_0)$, $a_1 \sim \pi(\cdot \mid s_1)$.
2. $$
   y = \begin{cases}
   r_0 + \gamma \, \mathrm{sg}[Q_{\phi}(s_1, a_1)], & \text{if } s_1 \neq \langle\text{term}\rangle \\
   r_0,                                               & \text{otherwise}
   \end{cases}
   $$
3. $g = \left(Q_{\phi}(s_0, a_0) - y\right) \nabla_{\phi} Q_{\phi}(s_0, a_0)$.
4. Update $\phi$ using $g$ with an optimizer.

</Algorithm>

<Algorithm title="Q-function approximation with k-step TD">

**while** not converged:

1. $s_0 \sim p_0$, set $t = 0$.
2. **for** $t = 0, \ldots, k-1$:
   - $a_t \sim \pi(\cdot \mid s_t)$
   - $(r_t, s_{t+1}) \sim p(\cdot, \cdot \mid s_t, a_t)$
3. $y = \sum_{t=0}^{k-1} \gamma^t r_t + \gamma^k \, \mathrm{sg}[Q_{\phi}(s_k, a_k)]$
4. $g = (Q_{\phi}(s_0, a_0) - y) \nabla_{\phi} Q_{\phi}(s_0, a_0)$
5. Update $\phi$ using $g$ with an optimizer.

</Algorithm>

### Policy Iteration Methods

介绍完 policy evaluation 之后，我们可以设计 policy iteration 算法，其基本思想是交替进行 policy evaluation 和 policy improvement:

$$
\begin{aligned}
&\text{Compute } V^{\pi^k} \text{ and } Q^{\pi^k} &\quad&\text{(Policy evaluation)} \\
&\pi^{k+1}(s) = \arg\max_{a \in \mathcal{A}} \mathbb{E}_{(r, s') \sim p(\cdot, \cdot \mid s, a)}\left[r + \gamma V^{\pi^k}(s') \mid s, a\right] &\quad&\text{(Policy improvement)}
\end{aligned}
$$

算法的正确性由如下定理保证

<Theorem>

**Theorem**: 考虑上述 policy iteration 算法，假设 $\gamma \in (0, 1)$, $|\mathcal{S}| < \infty$, $|\mathcal{A}| < \infty$ 以及 $|r| \leq R < \infty$, a.s. 那么：

$$
V^{\pi^{k+1}} \geq V^{\pi^k}, \quad k = 1, 2, \dots
$$

并且存在 $K \in \mathbb{N}$ 满足 $V^{\pi^K} = V^*$. 也就是说，有限步之后就可以得到最优策略.

</Theorem>
