speculative decoding 使用了两个模型来加速推理

1. draft model ($M_d$) , 用于生成一个 candidate sequence
2. target model ($M_t$), 用于对 candidate sequence 进行验证

工作原理：

1. drafting: 给定一个 prompt $x_{1:n}$, draft model 基于 prompt 生成一个包含 $K$ 个 token 的 candidate sequence $y_1,\dots,y_k$
2. verification: target model 对 输入 $\{(x_{1:n}, y_{1:k})\}_{k=1}^K$ 进行一次 forward 计算每个 token 的可能性
3. acceptance/rejection: 顺序检查 candidate sequence 中的每一个 token, 如果一个 token 不是概率最大的 token, 则该 token 与后续所有的 token 都会被丢弃
4. correction: 对于第一个被丢弃的 token, 使用 target model 生成的 token 来替换该 token, 然后调用 draft model 进一步生成
