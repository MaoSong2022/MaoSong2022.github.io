## Data

1. environment: 比如 code repository, web browser 等，让模型能够理解不同的环境
2. tools: 可以提供信息或者改变环境，让模型使用不同的工具
3. verifier: 比如代码的单元测试，proof 的 math checker, DOM scripts 等，训练模型完成不同的目标

关键：environment 和 verifier 的 diversity

verifier: 保证 我们有一些 false positive 和 false negatives

Mach-Mind-4-Flash [@Mach-Mind-4-Flash] 介绍了 EnvScaling, 来提高 multi-turn agentic RL 的表现。
