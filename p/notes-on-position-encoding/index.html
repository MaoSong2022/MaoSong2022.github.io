жң¬ж–ҮеүҚеҚҠйғЁеҲҶеҸӮиҖғ еҸӮиҖғж–ҮзҢ®1пјҢжҺЁиҚҗеӨ§е®¶зңӢеҚҡе®ўеҺҹж–ҮгҖӮ

Position encodingжҖ»з»“

еңЁ дёҠдёҖзҜҮblog дёӯ, жҲ‘д»¬д»Ӣз»ҚдәҶ Attention зҡ„дёӨдёӘжҖ§иҙЁпјҢд№ҹе°ұжҳҜеңЁдёҚеҠ  position encoding зҡ„жғ…еҶөдёӢпјҢAttention еҜ№дәҺ query жҳҜ permutation equivariant зҡ„пјҢеҜ№дәҺ key е’Ң value жҳҜ permutation invariant зҡ„гҖӮ

дҪҶжҳҜвҖңжҲ‘зҲұдҪ вҖқе’ҢвҖңдҪ зҲұжҲ‘вҖқиҝҷдёӨеҸҘиҜқжүҖиЎЁзӨәзҡ„еҗ«д№үеә”иҜҘжҳҜдёҚдёҖж ·зҡ„пјҢжҲ‘д»¬е°ҶиҝҷдёӨеҸҘиҜқдҪңдёә key е’Ң value зҡ„ж—¶еҖҷпјҢжҲ‘д»¬еҸ‘зҺ°жЁЎеһӢзҡ„иҫ“еҮәжҳҜдёҖиҮҙзҡ„пјҢиҝҷжҳҫз„¶жҳҜдёҚиғҪжҺҘеҸ—зҡ„гҖӮеӣ жӯӨпјҢжҲ‘д»¬е°ұйңҖиҰҒеҠ е…Ҙ position encodingпјҢи®©жЁЎеһӢеӯҰд№ еҲ°иҜӯеәҸдҝЎжҒҜпјҢд»ҺиҖҢжҳҺзҷҪдёҚеҗҢзҡ„иҜӯеәҸжңүдёҚеҗҢзҡ„еҗ«д№үгҖӮ

import torch
import torch.nn as nn
from transformers import AutoTokenizer, AutoModel

model_id = "meta-llama/Llama-3.2-1B"
tok = AutoTokenizer.from_pretrained(model_id)
model = AutoModel.from_pretrained(model_id)

text = "The dog chased another dog"
tokens = tok(text, return_tensors="pt")["input_ids"]
embeddings = model.embed_tokens(tokens)
hdim = embeddings.shape[-1]

W_q = nn.Linear(hdim, hdim, bias=False)
W_k = nn.Linear(hdim, hdim, bias=False)
W_v = nn.Linear(hdim, hdim, bias=False)
mha = nn.MultiheadAttention(embed_dim=hdim, num_heads=4, batch_first=True)

with torch.no_grad():
    for param in mha.parameters():
        nn.init.normal_(param, std=0.1) # Initialize weights to be non-negligible

output, _ = mha(W_q(embeddings), W_k(embeddings), W_v(embeddings))

dog1_out = output[0, 2]
dog2_out = output[0, 5]
print(f"Dog output identical?: {torch.allclose(dog1_out, dog2_out, atol=1e-6)}") #True

Position encoding еҸҜд»ҘеҲҶдёәз»қеҜ№дҪҚзҪ®зј–з Ғ (absolute position encoding, APE)пјҢзӣёеҜ№дҪҚзҪ®зј–з Ғ (relative position encoding, RPE) д»ҘеҸҠеҸҜеӯҰд№ зҡ„дҪҚзҪ®зј–з ҒгҖӮеҸҜеӯҰд№ дҪҚзҪ®зј–з Ғдё»иҰҒжҳҜ BERT зұ»зҡ„жЁЎеһӢеңЁдҪҝз”ЁпјҢе…¶и®ӯз»ғжҲҗжң¬жҜ”иҫғй«ҳпјҢжң¬ж–ҮдёҚеҒҡи®Ёи®әгҖӮз»қеҜ№дҪҚзҪ®зј–з ҒжҳҜеҺҹе§Ӣ transformer йҮҢжҸҗеҮәзҡ„зј–з ҒжЁЎејҸпјҢзҺ°еңЁзҡ„еӨ§еӨҡж•°еҹәдәҺ transformer жЁЎеһӢдҪҝз”Ёзҡ„йғҪжҳҜзӣёеҜ№дҪҚзҪ®зј–з ҒгҖӮ

жң¬ж–ҮдёӯпјҢжҲ‘д»¬е…Ҳд»Ӣз»ҚдҪҚзҪ®зј–з Ғеә”иҜҘе…·жңүзҡ„жҖ§иҙЁпјҢз„¶еҗҺжҲ‘д»¬еҲҶеҲ«д»Ӣз»Қз»қеҜ№дҪҚзҪ®зј–з Ғе’ҢзӣёеҜ№дҪҚзҪ®зј–з ҒпјҢжҲ‘д»¬е°ҶзқҖйҮҚе…іжіЁиӢҸеү‘жһ—иҖҒеёҲжҸҗеҮәжқҘзҡ„ RoPEгҖӮ

дҪҚзҪ®зј–з Ғ

еңЁд»Ӣз»ҚдҪҚзҪ®зј–з Ғд№ӢеүҚпјҢжҲ‘д»¬йҰ–е…Ҳеә”иҜҘе…іжіЁдҪҚзҪ®зј–з Ғзҡ„жҖ§иҙЁпјҢдҪҚзҪ®зј–з Ғзҡ„зӣ®ж ҮжҳҜдёәиҫ“е…Ҙзҡ„ token embedding еўһеҠ дҪҚзҪ®дҝЎжҒҜпјҢйӮЈд№ҲзҗҶжғізҡ„дҪҚзҪ®зј–з Ғеә”иҜҘжҳҜжҖҺд№Ҳж ·зҡ„е‘ўпјҹ

жҲ‘д»¬иҝҷйҮҢзӣҙжҺҘеј•з”Ё еҸӮиҖғж–ҮзҢ®1 дёӯз»ҷе®ҡзҡ„жҖ§иҙЁпјҡ

  1. жҖ§иҙЁ 1: token sequence дёӯжҜҸдёӘдҪҚзҪ®зҡ„дҪҚзҪ®зј–з ҒйғҪжҳҜе”ҜдёҖзҡ„гҖӮиҝҷдёӘеҫҲеҘҪзҗҶи§ЈпјҢеҰӮжһңдёҚе”ҜдёҖзҡ„иҜқпјҢйӮЈд№Ҳж №жҚ®еүҚйқўжҺЁеҜјзҡ„жҖ§иҙЁпјҢиҝҷдёӨдёӘдҪҚзҪ®зҡ„ attention иҫ“еҮәе°ұе®Ңе…ЁдёҖиҮҙдәҶ
  2. жҖ§иҙЁ 2: зәҝжҖ§зӣёе…іжҖ§гҖӮд№ҹе°ұжҳҜиҜҙпјҢеҰӮжһңжҲ‘д»¬зҹҘйҒ“дәҶдҪҚзҪ® $p$ еӨ„зҡ„дҪҚзҪ®зј–з ҒпјҢйӮЈд№ҲзҗҶжғіжғ…еҶөдёӢпјҢжҲ‘д»¬еә”иҜҘиғҪжҜ”иҫғз®ҖеҚ•ең°еҫ—еҲ° $p+k$ еӨ„зҡ„дҪҚзҪ®зј–з ҒпјҢзҗҶжғіжғ…еҶөдёӢпјҢжҲ‘д»¬еә”иҜҘжңү $PE(p+k)=W_kPE(p)$.
  3. жҖ§иҙЁ 3: жіӣеҢ–еҲ°й•ҝдёҠдёӢж–ҮдёӯеҺ»гҖӮжҲ‘д»¬еёҢжңӣдҪҚзҪ®зј–з ҒдёҚд»…еңЁ 8K зҡ„дёҠдёӢж–Үиө·дҪңз”ЁпјҢиҝҳеёҢжңӣдҪҚзҪ®зј–з ҒиғҪеӨҹжіӣеҢ–еҲ° 32K зҡ„дёҠдёӢж–Ү
  4. жҖ§иҙЁ 4: з”ҹжҲҗжЁЎејҸжҳҜеӣәе®ҡзҡ„гҖӮеӣәе®ҡзҡ„жЁЎејҸжңүеҠ©дәҺжЁЎеһӢжӣҙеҘҪең°еӯҰд№ дҪҚзҪ®зӣёе…ізҡ„дҝЎжҒҜ
  5. жҖ§иҙЁ 5: еҸҜд»Ҙжү©еұ•еҲ°еӨҡз»ҙгҖӮжҲ‘д»¬еёҢжңӣдҪҚзҪ®зј–з ҒеҸҜд»Ҙд»Һж–Үжң¬жү©еұ•еҲ°еӣҫзүҮеҶҚеҲ°и§Ҷйў‘пјҢд№ҹе°ұжҳҜд»Һ $1D$ еҲ° $nD$.

з»қеҜ№дҪҚзҪ®зј–з Ғ

з»қеҜ№дҪҚзҪ®зј–з Ғдҫқз…§е…¶еҗҚз§°пјҢе…¶жҖқжғіе°ұжҳҜдёәжҜҸдёӘдҪҚзҪ®зҡ„ token еҲҶй…ҚдёҖдёӘеӣәе®ҡзҡ„дҪҚзҪ®дҝЎжҒҜпјҢд№ҹе°ұжҳҜеҜ№дәҺиҫ“е…Ҙзҡ„ hidden states $\bm{x}=[\bm{x}_1,\dots,\ \bm{x}_m]\in\mathbb{R}^{m\times d}$, жҲ‘д»¬жңү

$$ \bm{x}_i' = \bm{x}_i + p_i, i=1,\dots, m $$

иҝҷйҮҢпјҢ$p_i\in\mathbb{R}^d$. жҲ‘д»¬зҡ„ attention е°ұеҸҳжҲҗдәҶ

$$ \mathrm{Attn}(X) = \mathrm{softmax}\left(\frac{(Q+P)(K+P)^T}{\sqrt{d}}\right)V\in\mathbb{R}^{m\times d} $$

иҝҷйҮҢ

$$ P = [p_1,\dots,p_m]\in\mathbb{R}^{m\times d}пјҢ Q= W_QX\in\mathbb{R}^{m\times d}, K=W_KX, V=W_VX\in\mathbb{R}^{n\times d} $$

ж•ҙж•°дҪҚзҪ®зј–з Ғ

дёҖдёӘжңҖз®ҖеҚ•зҡ„жғіжі•е°ұжҳҜжҲ‘д»¬дҪҝз”ЁжӯЈж•ҙж•°жқҘж Үи®° token жүҖеңЁзҡ„дҪҚзҪ®пјҢд№ҹе°ұжҳҜ

$$ PE(i) = [i, \dots, i]=i\mathbf{1}_{d\times 1}\in\mathbb{R}^d,\ i=1,\dots,m $$

еҸҜд»ҘзңӢеҲ°пјҢиҝҷдёӘз®ҖеҚ•зҡ„и®ҫи®Ўж»Ўи¶іжҖ§иҙЁ 1пјҢжҖ§иҙЁ 2пјҢжҖ§иҙЁ 3пјҢжҖ§иҙЁ 4.

дҪҶжҳҜпјҢжіЁж„ҸеҲ° attention зҡ„иҫ“е…Ҙ $X$ йҖҡеёёжҳҜз»ҸиҝҮ Layer Normalization еӨ„зҗҶиҝҮеҗҺзҡ„пјҢеӣ жӯӨе…¶жҢүеҲ—з¬ҰеҗҲжӯЈжҖҒеҲҶеёғпјҢе№¶дё”еқҮеҖје’Ңж–№е·®дёҖиҲ¬иҫғе°ҸгҖӮеҪ“жҲ‘д»¬еҠ дёҠж•ҙж•°дҪҚзҪ®зј–з Ғд№ӢеҗҺпјҢе…¶ token жң¬иә«зҡ„дҝЎжҒҜе°ұдјҡиў«жұЎжҹ“пјҢд№ҹе°ұжҳҜдҝЎеҷӘжҜ”йқһеёёдҪҺгҖӮдёҖдёӘи§ЈеҶіж–№жі•е°ұжҳҜжҲ‘д»¬еҜ№ $PE(i)$ иҝӣиЎҢ normalizationпјҢеҚі

$$ PE(i)' = \frac{1}{m}PE(i) = \frac{i}{m}\mathbf{1}_{d\times 1} $$

зҺ°еңЁжүҖжңүзҡ„дҪҚзҪ®зј–з Ғзҡ„еҖјйғҪжҜ”иҫғе°ҸпјҢдҪҶжҳҜжҲ‘д»¬еҸ‘зҺ°ж–°зҡ„дҪҚзҪ®зј–з ҒдёҚж»Ўи¶іжҖ§иҙЁ 2 дәҶпјҢиҝҷжҳҜеӣ дёәзҺ°еңЁдҪҚзҪ®зј–з Ғиҝҳе’Ң sequence й•ҝеәҰжңүе…іпјҢжҲ‘д»¬д»ҺдҪҚзҪ® $p$ еҲ°дҪҚзҪ® $p+k$ дёҚд»…еҸ–еҶідәҺ $k$ иҝҳеҸ–еҶідәҺ sequence й•ҝеәҰ $m$

дәҢиҝӣеҲ¶дҪҚзҪ®зј–з Ғ

ж—ўз„¶ж•ҙж•°дҪҚзҪ®зј–з Ғзҡ„дё»иҰҒй—®йўҳжҳҜеҜ№иҫ“е…ҘеҪұе“ҚеӨӘеӨ§пјҢжҲ‘д»¬иғҪеҗҰжүҫдёҖдёӘдёҚеҪұе“Қиҫ“е…Ҙзҡ„ж•ҙж•°дҪҚзҪ®зј–з Ғж–№ејҸе‘ўпјҹ еҸӮиҖғж–ҮзҢ®1 жҸҗеҮәдәҶдәҢиҝӣеҲ¶дҪҚзҪ®зј–з ҒпјҢеӣ дёәжҜҸдёӘ token жҳҜ $d$ з»ҙзҡ„пјҢеӣ жӯӨжҲ‘д»¬еҸҜд»ҘдҪҝз”Ё $d$ дҪҚдәҢиҝӣеҲ¶жқҘиЎЁзӨә $i$. жҜ”еҰӮиҜҙпјҢеҪ“ $d=3$, $m=4$ ж—¶пјҢжҲ‘д»¬зҡ„дҪҚзҪ®зј–з ҒеҲҶеҲ«дёә

$$ PE(0) =p_{(000)_2} = [0, 0, 0],\ PE(1) =p_{(001)_2}= [0, 0, 1],\ PE(2) =p_{(010)_2} = [0, 1, 0],\ PE(3) =p_{(011)_2} = [0, 1, 1] $$

зҺ°еңЁпјҢжҲ‘д»¬дәҢиҝӣеҲ¶дҪҚзҪ®зј–з Ғж»Ўи¶іжҖ§иҙЁ 1пјҢжҖ§иҙЁ 2. еҜ№дәҺжҖ§иҙЁ 3пјҢз”ұдәҺ $d$ дҪҚдәҢиҝӣеҲ¶зҡ„иЎЁзӨәиҢғеӣҙдёә $[0, 2^d-1]$пјҢеӣ жӯӨе…¶жіӣеҢ–жҖ§еҸ—еҲ° $d$ зҡ„еҪұе“ҚгҖӮ

еҸӮиҖғж–ҮзҢ®1 з”»еҮәдәҶдёҚеҗҢдҪҚзҪ®зҡ„еҖјзҡ„еҸҳеҢ–жғ…еҶөгҖӮжҲ‘д»¬иҝҷйҮҢд№ҹжЁЎд»ҝз»ҳеҲ¶еҮәзұ»дјјзҡ„жӣІзәҝеӣҫ

жҲ‘д»¬еҸ‘зҺ°пјҢдәҢиҝӣеҲ¶дҪҚзҪ®зј–з Ғй«ҳдҪҚпјҢд№ҹе°ұжҳҜ $PE(i){d}$ зҡ„еҸҳеҢ–еҫҲж…ўпјҢиҖҢдҪҺдҪҚпјҢд№ҹе°ұжҳҜ $PE(i){0}$ еҸҳеҢ–еҫҲеҝ«пјҢ

дәҢиҝӣеҲ¶дҪҚзҪ®зј–з Ғи§ЈеҶідәҶж•ҙж•°дҪҚзҪ®зј–з Ғзҡ„дҝЎеҷӘжҜ”иҝҮдҪҺе’ҢзәҝжҖ§зӣёе…іжҖ§гҖӮдҪҶжҳҜе…¶й—®йўҳжҳҜе…¶еҜ№дёҚеҗҢдҪҚзҪ®зҡ„ token embedding дә§з”ҹзҡ„еҪұе“ҚжҳҜдёҚдёҖж ·зҡ„гҖӮжҜ”еҰӮдҪҚзҪ® 1 е’ҢдҪҚзҪ® 2 зҡ„зӣёеҗҢзҡ„ token embedding д№Ӣй—ҙзҡ„еҢәеҲ«жҳҜпјҡ

$$ (\bm{x}_2 + PE(2)) - (\bm{x}_1 + PE(1)) = (\bm{x}_2-\bm{x}_1)+ [0, 1, -1] $$

дёҖиҲ¬жқҘиҜҙ, $\bm{x}_2-\bm{x}_1$ жҜ”иҫғе°ҸпјҢеӣ жӯӨдҪҝз”ЁдәҢиҝӣеҲ¶дҪҚзҪ®зј–з Ғзҡ„й—®йўҳжҳҜиҫ“е…ҘдҪҚзҪ®зҡ„еҫ®е°ҸеҸҳеҢ–пјҲеўһеҠ дёҖдёӘ token жҲ–еҮҸе°‘дёҖдёӘ tokenпјүйғҪдјҡеҜ№жңҖз»Ҳз»“жһңдә§з”ҹе·ЁеӨ§еҪұе“ҚгҖӮеӣ жӯӨпјҢжҲ‘д»¬йңҖиҰҒжғіеҠһжі•и§ЈеҶіиҝҷдёӘй—®йўҳгҖӮ

Sinusoidal

еүҚйқўжҸҗеҲ°дәҢиҝӣеҲ¶дҪҚзҪ®зј–з Ғзҡ„й—®йўҳжҳҜзӣёйӮ» token д№Ӣй—ҙеҸҳеҢ–еӨӘеӨ§пјҢдёҚеӨҹе…үж»‘гҖӮеӣ жӯӨжҲ‘д»¬жғіиҰҒеўһеҠ дёҖдёӘе…үж»‘жҖ§иҙЁпјҢд№ҹе°ұжҳҜиҜҙжҲ‘д»¬еёҢжңӣпјҡ

  1. дҪҚзҪ®зј–з ҒеҖјеңЁ $[-1, 1]$ д№Ӣй—ҙпјҢйҳІжӯўеҜ№ token embedding дә§з”ҹеҪұе“Қ
  2. зӣёйӮ» token зҡ„дҪҚзҪ®зј–з Ғе°ҪеҸҜиғҪзӣёиҝ‘пјҢеҚі $|PE(k+p)-PE(p)| \leq \delta |k|$, е…¶дёӯ $\delta>0$ жҳҜдёҖдёӘжҜ”иҫғе°Ҹзҡ„ж•°гҖӮ
  3. дёҺдәҢиҝӣеҲ¶дёҖж ·пјҢй«ҳдҪҚзҡ„еҸҳеҢ–жҜ”иҫғж…ўпјҢдҪҺдҪҚзҡ„еҸҳеҢ–жҜ”иҫғеҝ«

дёҖдёӘжғіжі•е°ұжҳҜеҲ©з”Ёдёүи§’еҮҪж•° $\sin$ жҲ–иҖ… $\cos$пјҢдёүи§’еҮҪж•°ж»Ўи¶іеүҚдёӨдёӘжҖ§иҙЁпјҢ еҜ№дәҺз¬¬дёүдёӘжҖ§иҙЁпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮжҺ§еҲ¶йў‘зҺҮжқҘж»Ўи¶ігҖӮиҝҷж ·жҲ‘д»¬еҫ—еҲ°зҡ„дҪҚзҪ®зј–з Ғе°ұе…·жңүеҰӮдёӢеҪўејҸпјҡ

$$ PE(p, i) = \sin\left(\frac{p}{\theta^{i/d}}\right) $$

е…¶дёӯ $\theta$ жҳҜжҲ‘д»¬зҡ„и¶…еҸӮж•°гҖӮ

жҲ‘д»¬зҺ°еңЁжқҘжҺЁеҜјдёҖдёӢдёҠйқўдҪҚзҪ®зј–з Ғзҡ„зәҝжҖ§зӣёе…іжҖ§пјҡ

$$ PE(p+k) = \sin\left(\frac{p+k}{\theta^{i/d}}\right)=PE(p)\cos\left(\frac{k}{\theta^{i/d}}\right) + \cos\left(\frac{p}{\theta^{i/d}}\right)\sin\left(\frac{k}{\theta^{i/d}}\right) $$

жҲ‘д»¬еҸ‘зҺ°пјҢ$\sin$ дҪҚзҪ®зј–з ҒдёҚж»Ўи¶ізәҝжҖ§зӣёе…іжҖ§гҖӮдҪҶжҳҜеҮәзҺ°зҡ„ $\cos$ з»ҷдәҶжҲ‘д»¬еҗҜеҸ‘пјҢд№ҹе°ұжҳҜжҲ‘д»¬еҸҜд»ҘеҗҢж—¶дҪҝз”Ё $\sin$ е’Ң $\cos$ жқҘе®ҢжҲҗдҪҚзҪ®зј–з ҒпјҢиҝҷд№ҹжҳҜеҺҹе§Ӣ transformer йҮҢжҸҗеҮәжқҘзҡ„ Sinusoidal дҪҚзҪ®зј–з ҒпјҢе…¶еҪўејҸдёәпјҡ

$$ \begin{aligned} PE(p, 2i) &= \sin\left(\frac{p}{\theta^{2i/d}}\right)\\ PE(p, 2i+1) &= \cos\left(\frac{p}{\theta^{2i/d}}\right) \end{aligned} $$

зҺ°еңЁпјҢи®° $\omega_i=1/\theta^{2i/d}$, жҲ‘д»¬еҶҚжҺЁеҜјдёҖдёӢзәҝжҖ§зӣёе…іжҖ§пјҢе°ұеҫ—еҲ°пјҡ

$$ \begin{aligned} \begin{bmatrix} PE(p+k, 2i)\\ PE(p+k, 2i+1)\\ \end{bmatrix}&=\begin{bmatrix} \sin \omega_i(p+k)\\ \cos \omega_i(p+k) \end{bmatrix}\\ &=\begin{bmatrix} \sin \omega_i(\omega_ip)\cos(\omega_ik)+\cos w_i(\omega_ip)\sin(\omega_ik)\\ \cos \omega_i(\omega_ip)\cos(\omega_ik)-\sin w_i(\omega_ip)\sin(\omega_ik) \end{bmatrix}\\ &= \begin{bmatrix} \cos(\omega_ik)& \sin(\omega_ik)\\ -\sin(\omega_ik)& \cos(\omega_ik) \end{bmatrix}\begin{bmatrix} PE(p, 2i)\\ PE(p, 2i+1)\\ \end{bmatrix} \end{aligned} $$

д№ҹе°ұжҳҜиҜҙпјҢSinusoidal дҪҚзҪ®зј–з Ғж»Ўи¶ізәҝжҖ§зӣёе…іжҖ§гҖӮеҜ№дәҺ Sinusoidal дҪҚзҪ®зј–з ҒжҲ‘д»¬д№ҹеҸҜд»ҘиҝӣиЎҢеҸҜи§ҶеҢ–пјҡ

зӣёеҜ№дҪҚзҪ®зј–з Ғ

еүҚйқўд»Ӣз»ҚдәҶз»қеҜ№дҪҚзҪ®зј–з ҒпјҢжҜҸдёӘдҪҚзҪ®зҡ„дҪҚзҪ®зј–з ҒжҳҜеӣәе®ҡзҡ„гҖӮдҪҶжҳҜз»қеҜ№дҪҚзҪ®зј–з Ғзҡ„й—®йўҳжҳҜпјҢжЁЎеһӢжҜ”иҫғйҡҫд»ҘеӯҰд№ зӣёеҜ№дҪҚзҪ®е…ізі»гҖӮ

дёҫдёӘдҫӢеӯҗпјҢжҲ‘д»¬жҸҗеҲ°дёҠдёӢж–Үж—¶пјҢйҖҡеёёдјҡдҪҝз”ЁвҖңдёҠдёҖиҠӮвҖқпјҢвҖңдёҠдёҖз« вҖқиҝҷдәӣиЎЁзӨәзӣёеҜ№дҪҚзҪ®е…ізі»зҡ„иҜҚгҖӮ

еӣ жӯӨпјҢжҲ‘д»¬еёҢжңӣи®©жЁЎеһӢеӯҰд№ зӣёеҜ№дҪҚзҪ®е…ізі»иҖҢдёҚжҳҜз»қеҜ№дҪҚзҪ®е…ізі»пјҢеӣ дёәзӣёеҜ№е…ізі»жӣҙз¬ҰеҗҲжҲ‘д»¬зҡ„и®ӨзҹҘгҖӮ

RoPE

RoPE з”ұиӢҸеү‘жһ—иҖҒеёҲжҸҗеҮәпјҢжңҖж—©еә”з”ЁдәҺ LLaMA жһ¶жһ„пјҲжІЎжңүзЎ®и®ӨпјүпјҢеҗҺз»ӯиў«еӨ§еӨҡж•°жЁЎеһӢжүҖйҮҮз”ЁгҖӮ

д№ӢеүҚзҡ„ PE еӨ§еӨҡж•°е…іжіЁдәҺеҠ жҖ§дҪҚзҪ®зј–з ҒпјҢд№ҹе°ұжҳҜеҒҮи®ҫдҪҚзҪ®зј–з Ғзҡ„еҪўејҸдёә $\bm{x}+\bm{p}$, еҹәдәҺиҝҷз§ҚеҒҮи®ҫпјҢе·Іжңүзҡ„е·ҘдҪңеҹәжң¬йғҪйӣҶдёӯдәҺдјҳеҢ–дёӢйқўзҡ„ Q е’Ң K зҡ„еҶ…з§Ҝ

$$ \langle f_q(\bm{x}_q, m), f_k(\bm{x}_k, n) \rangle $$

иҝҷйҮҢ $f_q(\bm{x}_q, m)=W_q(\bm{x}_q+\bm{p}_m)$, $f_k(\bm{x}_k, n)=W_k(\bm{x}_k+ \bm{p}_n)$.

иҖҢ RoPE йҮҢйқўпјҢдҪңиҖ…дҪҝз”ЁдәҶдёҖдёӘдёҚеҗҢзҡ„еҒҮи®ҫпјҡ еҒҮи®ҫеҶ…з§Ҝеә”иҜҘд»…еҢ…еҗ«дёӨиҖ…зҡ„зӣёеҜ№дҝЎжҒҜпјҢд№ҹе°ұжҳҜ

$$ \langle f_q(\bm{x}_q, m), f_q(\bm{x}_k, n)\rangle := g(\bm{x}_q,\bm{x}_k, m-n) $$

иҝҷйҮҢзҡ„ $f$ е’Ң $g$ йғҪжҳҜжңӘзҹҘеҮҪж•°гҖӮжҲ‘д»¬зҡ„зӣ®ж Үе°ұжҳҜд»ҺиҝҷдёӘе…¬ејҸдёӯжҺЁеҜјеҮәдёҖдёӘеҗҲйҖӮзҡ„дҪҚзҪ®зј–з ҒеҮәжқҘгҖӮ

дёҚеӨұдёҖиҲ¬жҖ§пјҢжҲ‘д»¬еҸҜд»ҘеҒҮи®ҫ

$$ f_q(\bm{x}_m,0) = \bm{x}_q,\quad f_q(\bm{x}_n, 0) = \bm{x}_k $$

иҝҷдёӘеҒҮи®ҫд»ЈиЎЁеҲқе§ӢжқЎд»¶дёӢпјҢжҲ‘д»¬дёҚеҜ№иҫ“е…ҘеҒҡд»»дҪ•ж”№еҸҳпјҢд№ҹе°ұжҳҜдёҚеўһеҠ дҪҚзҪ®дҝЎжҒҜгҖӮ

2D жҺЁеҜј

дёҺ RoPE дёҖж ·пјҢжҲ‘д»¬зӣҙжҺҘдҪҝз”ЁеӨҚе№ійқўжқҘиҝӣиЎҢжҺЁеҜјгҖӮ

жҲ‘д»¬еҒҮи®ҫ $d=2$, жіЁж„ҸеҲ°дәҢз»ҙе№ійқўдёҠзҡ„жҜҸдёӘзӮ№йғҪеҸҜд»ҘиЎЁзӨәдёәеҰӮдёӢеҪўејҸ

$$ \bm{z} = (x,y) = re^{i\theta} $$

е…¶дёӯ ($\mathrm{atan2}$ е®ҡд№үеҸӮиҖғ з»ҙеҹәзҷҫз§‘)

$$ r = \|\bm{z}\|_2 = \sqrt{x^2+y^2}\in\mathbb{R},\quad \theta = \mathrm{atan2}(y, x)\in\mathbb{R}, $$

зҺ°еңЁпјҢеҜ№дәҺдёүдёӘеҗ‘йҮҸ $f_q(\bm{x}_q, m)$, $f_q(\bm{x}_k, n)$, $g(\bm{x}_q,\bm{x}_k, m-n)$ жҲ‘д»¬еҸҜд»ҘеҶҷеҮәе…¶жһҒеқҗж ҮеҪўејҸпјҡ

$$ \begin{aligned} f_q(\bm{x}_q,m) &:= r_q(\bm{x}_q,m)e^{i\theta_q(\bm{x}_q,m)}\\ f_k(\bm{x}_k, n) &:= r_k(\bm{x}_k, n)e^{i\theta_k(\bm{x}_k, n)}\\ g(\bm{x}_q,\bm{x}_k, m-n) &:= r_g(\bm{x}_q,\bm{x}_k, m-n)e^{i\theta_g(\bm{x}_q,\bm{x}_k, m-n)} \end{aligned} $$

жҲ‘д»¬и®Ўз®—еҶ…з§Ҝе№¶жҜ”иҫғеҗҢзұ»йЎ№еҫ—еҲ°пјҡ

$$ \begin{aligned} r_g(\bm{x}_q,\bm{x}_k, m-n) &:= r_q(\bm{x}_q,m)r_k(\bm{x}_k, n)\\ \theta_g(\bm{x}_q,\bm{x}_k, m-n) &:= \theta_q(\bm{x}_q,m)-\theta_k(\bm{x}_k, n) \end{aligned}\tag{3} $$

жҲ‘д»¬жҺҘдёӢжқҘеҲҶеҲ«жҺЁеҜј $r_g(\bm{x}_q,\bm{x}_k, m-n)$ е’Ң $\theta_g(\bm{x}_q,\bm{x}_k, m-n)$ зҡ„еҪўејҸ

$r_g(\bm{x}_q,\bm{x}_k, m-n)$

жҲ‘д»¬д»Ө $m=n=0$ еҸҜд»Ҙеҫ—еҲ°еҲқе§ӢжқЎд»¶

$$ r_g(\bm{x}_q,\bm{x}_k, 0) = r_q(\bm{x}_q,0)r_k(\bm{x}_k, 0)=\|\bm{q}\|_2\|\bm{k}\|_2 $$

жҲ‘д»¬еҶҚд»Ө $n=0$,еҫ—еҲ°

$$ r_g(\bm{x}_q,\bm{x}_k, m) = r_q(\bm{x}_q,m)r_k(\bm{x}_k, 0)=r_q(\bm{x}_q,m)\|\bm{k}\|_2=\frac{r_g(\bm{x}_q,\bm{x}_k, m-n)}{r_k(\bm{x}_k, n)}\|\bm{k}\|_2 $$

иҝҷйҮҢжңҖеҗҺдёҖдёӘзӯүејҸеёҰе…ҘдәҶеҺҹе§ӢзӯүејҸ (3)пјҢжіЁж„ҸеҲ°е·Ұдҫ§дёҺ $n$ ж— е…іпјҢеӣ жӯӨеҸідҫ§жҲ‘д»¬йҖүеҸ– $n=1$, еҫ—еҲ°

$$ r_g(\bm{x}_q,\bm{x}_k, m) = \frac{r_g(\bm{x}_q,\bm{x}_k, m-1)}{r_k(\bm{x}_k, 1)}\|\bm{k}\|_2 =\cdots= r_g(\bm{x}_q,\bm{x}_k, 0)\left(\frac{\|\bm{k}\|_2 }{r_k(\bm{x}_k, 1)}\right)^{m+1} $$

д»Ө $m=0$ жҲ‘д»¬жңү

$$ r_k(\bm{x}_k, 1) = \|\bm{k}\|_2. $$

еӣ жӯӨжҲ‘д»¬жңҖз»Ҳзҡ„иЎЁиҫҫејҸдёәпјҡ

$$ r_g(\bm{x}_q,\bm{x}_k, m) = r_g(\bm{x}_q,\bm{x}_k, 0) = \|\bm{q}\|_2\|\bm{k}\|_2. $$

е№¶дё”пјҢйҖҡиҝҮеҲҶеҲ«и®ҫзҪ® $m=0$ д»ҘеҸҠ $n=0$ жҲ‘д»¬иҝҳеҸҜд»Ҙеҫ—еҲ°

$$ r_q(\bm{x}_q,m) = \|\bm{q}\|_2,\quad r_k(\bm{x}_k, n) = \|\bm{k}\|_2 $$

$\theta_g(\bm{x}_q,\bm{x}_k, m-n)$

д»Ө $m=n=0$, жҲ‘д»¬еҫ—еҲ°еҲқе§ӢжқЎд»¶

$$ \theta_g(\bm{x}_q,\bm{x}_k, 0) = \theta_q(\bm{x}_q,0)-\theta_k(\bm{x}_k, 0)=\theta_q-\theta_k $$

д»Ө $n=1$, жҲ‘д»¬жңү

$$ \begin{aligned} \theta_g(\bm{x}_q,\bm{x}_k, m-1) &= \theta_q(\bm{x}_q,m)-\theta_k(\bm{x}_k, 1)\\ &=\theta_g(\bm{x}_q,\bm{x}_k, m-n) + \theta_k(\bm{x}_k, n)-\theta_k(\bm{x}_k, 1) \end{aligned} $$

иҝҷйҮҢжҲ‘д»¬еёҰе…ҘдәҶе…¬ејҸ (3)пјҢжіЁж„ҸеҲ°е…¬ејҸе·Ұиҫ№дёҺ $n$ ж— е…іпјҢеӣ жӯӨеңЁе…¬ејҸеҸідҫ§жҲ‘д»¬д»Ө $n=0$, еҫ—еҲ°

$$ \theta_g(\bm{x}_q,\bm{x}_k, m-1) = \theta_g(\bm{x}_q,\bm{x}_k, m)+ \theta_k(\bm{x}_k, 0)-\theta_k(\bm{x}_k, 1) $$

еҲҶеҲ«д»Ө $m=1,2,\dots$ е№¶зӣёеҠ иҝҷдәӣзӯүејҸпјҢжҲ‘д»¬еҫ—еҲ°

$$ \theta_g(\bm{x}_q,\bm{x}_k, 0) = \theta_g(\bm{x}_q,\bm{x}_k, m) + m(\theta_k(\bm{x}_k, 0)-\theta_k(\bm{x}_k, 1)) $$

еҚі

$$ \theta_g(\bm{x}_q,\bm{x}_k, m) = m(\theta_k(\bm{x}_k, 1)-\theta_k(\bm{x}_k, 0))+(\theta_q-\theta_k)\tag{4} $$

жіЁж„ҸеҲ°

$$ \theta_g(\bm{x}_q,\bm{x}_k, m) = \theta_q(\bm{x}_q,m)-\theta_k(\bm{x}_k, 0)=\theta_q(\bm{x}_q,m)-\theta_k $$

еёҰе…ҘдёҠејҸжҲ‘д»¬е°ұеҫ—еҲ°

$$ \theta_q(\bm{x}_q,m) = m(\theta_k(\bm{x}_k, 1)-\theta_k(\bm{x}_k, 0))+\theta_q $$

еңЁ (4) ејҸдёӯеҶҚд»Ө $m=m-n$пјҢе№¶еёҰе…Ҙ $\theta_q(\bm{x}_q,m)$ е°ұжңү

$$ \theta_k(\bm{x}_k,n) = n(\theta_k(\bm{x}_k, 1)-\theta_k(\bm{x}_k, 0))+\theta_k $$

жұҮжҖ»

жңҖеҗҺпјҢжҲ‘д»¬е°Ҷд»ҘдёҠз»“жһңж”ҫеңЁдёҖиө·пјҢе°ұеҫ—еҲ°

$$ f_q(\bm{x}_q,m) = \bm{q}e^{im\theta}, f_v(\bm{x}_k,m) = \bm{k}e^{in\theta} $$

иҝҷйҮҢ $\theta=\theta_k(\bm{x}_k, 1)-\theta_k(\bm{x}_k, 0)$ жҳҜдёҖдёӘи¶…еҸӮж•°пјҢз”ЁдәҺжҺ§еҲ¶йў‘зҺҮгҖӮ

жҲ‘д»¬и®°

$$ R_{\theta,m} = \begin{bmatrix} \cos m\theta & -\sin m\theta\\ \sin m\theta & \cos m\theta \end{bmatrix} $$

еҲҷжҲ‘д»¬жңүпјҡ

$$ f_q(\bm{x}_q,m) = R_{\theta,m}\bm{q}, f_v(\bm{x}_k,m) = R_{\theta,n}\bm{k}. $$

е№¶дё”

$$ \langle f_q(\bm{x}_q,m), f_v(\bm{x}_k,m)\rangle = \bm{q}^TR_{\theta,m-n}\bm{k} \tag{5} $$

еӨҡз»ҙжү©еұ•

дёҠйқўжҳҜ 2D зҡ„жғ…еҶөпјҢеҜ№дәҺеӨҡз»ҙжғ…еҶөпјҢиӢҸеү‘жһ—иҖҒеёҲйҖҡиҝҮе°ҶдёӨдёӘе…ғзҙ з»„еҜ№пјҢз„¶еҗҺеҲҶеҲ«иҝӣиЎҢеӨ„зҗҶпјҢеҫ—еҲ°дәҶеӨҡз»ҙзҡ„жғ…еҪўпјҡ

$$ R_{\theta,m}^d = \begin{bmatrix} R_{\theta_1,m} & & && & \\ & & R_{\theta_2,m} & & & \\ &&&& \ddots & \\ &&&& & R_{\theta_{d/2},m} \end{bmatrix}\in\mathbb{R}^{d\times d} $$

жҲ‘д»¬еҸҜд»ҘйӘҢиҜҒе…¬ејҸ (5) д»Қз„¶жҳҜжҲҗз«Ӣзҡ„гҖӮ

RoPE зҡ„иҝңзЁӢиЎ°еҮҸжҖ§иҙЁ

жҲ‘д»¬жҺҘдёӢжқҘзңӢдёҖдёӢз»“жһңдёҺзӣёеҜ№и·қзҰ» $m-n$ д№Ӣй—ҙзҡ„е…ізі»пјҢ жіЁж„ҸеҲ°

$$ \langle f_q(\bm{x}_q,m), f_v(\bm{x}_k,m)\rangle = \bm{q}^TR_{\theta,m-n}\bm{k} = \sum_{i=1}^{d/2} \bm{q}_i^TR_{\theta, m-n}\bm{k}_i $$

иҝҷйҮҢ $\bm{q}i=[q{2i},q_{2i+1}]^T$, $\bm{k}i=[k{2i},k_{2i+1}]^T$ еҲҶеҲ«жҳҜеҜ№еә”зҡ„ pairпјҢжҲ‘д»¬иҖғиҷ‘е…¶дёӯдёҖдёӘеҲҶйҮҸпјҢдёҚеҰЁеҒҮи®ҫ $|\bm{q}|_2=|\bm{k}|_2=1$, жҲ‘д»¬жңү

$$ \begin{aligned} \bm{q}_i^TR_{\theta, m-n}\bm{k}_i &\leq \bm{q}_i^TR_{\theta, m-n}\bm{q}_i\\ &= \bm{q}_i^T\left(\frac{R_{\theta, m-n}+R_{\theta, m-n}^T}{2}\right)\bm{q}_i\\ &\leq \lambda_{\max}\left(\frac{R_{\theta, m-n}+R_{\theta, m-n}^T}{2}\right) \\ &= \cos (m-n)\theta_i \end{aligned} $$

е…¶дёӯпјҢз¬¬дёҖдёӘдёҚзӯүејҸжҳҜеӣ дёә дёӨдёӘеҗ‘йҮҸзӣёзӯүж—¶е…¶еҶ…з§ҜжңҖеӨ§пјҢз¬¬дәҢдёӘдёҚзӯүејҸжҳҜз”ұдәҺдәҢж¬ЎеһӢжңҖеӨ§еҖјдёәзҹ©йҳөзҡ„зү№еҫҒеҖјгҖӮ

иҝҷж ·жҲ‘д»¬е°ұжңү

$$ \langle f_q(\bm{x}_q,m), f_v(\bm{x}_k,m)\rangle \leq \sum_{i=1}^{d/2}\cos (m-n)\theta_i. $$

жҲ‘д»¬еҸҜд»Ҙз®ҖеҚ•з”»еҮәеҜ№еә”зҡ„жӣІзәҝпјҡ

иҝҷйҮҢй’ҲеҜ№дёҚеҗҢзҡ„й…ҚзҪ®пјҢз»“жһңд№ҹдјҡз•Ҙеҫ®дёҚеҗҢпјҢе…·дҪ“еҲҶжһҗеҸҜд»ҘеҸӮеҠ зҹҘд№Һеӣһзӯ” RoPEзҡ„иҝңи·қзҰ»иЎ°еҮҸ

def apply_rotary_pos_emb(x: torch.Tensor, sin: torch.Tensor, cos: torch.Tensor) -> torch.Tensor:
    x_even = x[..., ::2]  # (seq_len, d_k_half)
    x_odd = x[..., 1::2]  # (seq_len, d_k_half)
    odds = cos * x_even - sin * x_odd  # (...,seq_len, d_k_half)
    evens = sin * x_even + cos * x_odd  # (...,seq_len, d_k_half)
    stacked = torch.stack((odds, evens), -2)  # (...,seq_len, 2, d_k_half)
    stacked_trans = rearrange(
        stacked, "... seq_len double d_k_half -> ... seq_len d_k_half double"
    )  # (...,seq_len, d_k_half, 2)
    out = rearrange(
        stacked_trans, "... seq_len d_k_half double -> ... seq_len (d_k_half double)"
    )  # (..., seq_len, d_k)
    return out

def precompute_freqs_cis(dim: int, end: int, theta: float = 10000.0):
    freqs = 1.0 / (theta ** (torch.arange(0, dim, 2)[: (dim // 2)].float() / dim))
    t = torch.arange(end, device=freqs.device)  # type: ignore
    freqs = torch.outer(t, freqs).float()  # type: ignore
    freqs_cis = torch.polar(torch.ones_like(freqs), freqs)  # complex64
    return freqs_cis

def reshape_for_broadcast(freqs_cis: torch.Tensor, x: torch.Tensor):
    ndim = x.ndim
    assert 0 <= 1 < ndim
    assert freqs_cis.shape == (x.shape[1], x.shape[-1])
    shape = [d if i == 1 or i == ndim - 1 else 1 for i, d in enumerate(x.shape)]
    return freqs_cis.view(*shape)


def apply_rotary_emb(
    xq: torch.Tensor,
    xk: torch.Tensor,
    freqs_cis: torch.Tensor,
) -> Tuple[torch.Tensor, torch.Tensor]:
    xq_ = torch.view_as_complex(xq.float().reshape(*xq.shape[:-1], -1, 2))
    xk_ = torch.view_as_complex(xk.float().reshape(*xk.shape[:-1], -1, 2))
    freqs_cis = reshape_for_broadcast(freqs_cis, xq_)
    xq_out = torch.view_as_real(xq_ * freqs_cis).flatten(3)
    xk_out = torch.view_as_real(xk_ * freqs_cis).flatten(3)
    return xq_out.type_as(xq), xk_out.type_as(xk)

def rotate_half(x):
    """Rotates half the hidden dims of the input."""
    x1 = x[..., : x.shape[-1] // 2]
    x2 = x[..., x.shape[-1] // 2 :]
    return torch.cat((-x2, x1), dim=-1)

def apply_rotary_pos_emb(q, k, cos, sin, position_ids=None, unsqueeze_dim=1):
    cos = cos.unsqueeze(unsqueeze_dim)
    sin = sin.unsqueeze(unsqueeze_dim)
    q_embed = (q * cos) + (rotate_half(q) * sin)
    k_embed = (k * cos) + (rotate_half(k) * sin)
    return q_embed, k_embed

class LlamaRotaryEmbedding(torch.nn.Module):
    def __init__(self, dim, max_position_embeddings=2048, base=10000, device=None):
        super().__init__()

        self.dim = dim
        self.max_position_embeddings = max_position_embeddings
        self.base = base
        inv_freq = 1.0 / (self.base ** (torch.arange(0, self.dim, 2).float().to(device) / self.dim))
        self.register_buffer("inv_freq", inv_freq)

        self.max_seq_len_cached = seq_len
        t = torch.arange(self.max_seq_len_cached, device=device, dtype=self.inv_freq.dtype)

        freqs = torch.einsum("i,j->ij", t, self.inv_freq)
        # Different from paper, but it uses a different permutation in order to obtain the same calculation
        emb = torch.cat((freqs, freqs), dim=-1)
        self.register_buffer("cos_cached", emb.cos()[None, None, :, :].to(dtype), persistent=False)
        self.register_buffer("sin_cached", emb.sin()[None, None, :, :].to(dtype), persistent=False)

    def forward(self, x, seq_len=None):
        # x: [bs, num_attention_heads, seq_len, head_size]
        if seq_len > self.max_seq_len_cached:
            self._set_cos_sin_cache(seq_len=seq_len, device=x.device, dtype=x.dtype)

        return (
            self.cos_cached[:, :, :seq_len, ...].to(dtype=x.dtype),
            self.sin_cached[:, :, :seq_len, ...].to(dtype=x.dtype),
        )
